# T2 Open Data Layer

T2 aggregates DerbySoft core systems with free and open global data sources to provide a unified intelligence layer for the hotel and travel industry.

## DerbySoft Core Systems

| System | Capability | MCP Name |
|--------|-----------|----------|
| Content Suite | Hotel content and media | `hotel-content-mcp` |
| Property Connector | Property sync and data onboarding | `property-connector-mcp` |
| Go | Direct distribution and marketing | `go-distribution-mcp` |
| Exchange | Distribution read/write | `exchange-distribution-mcp` |
| BI Platform | Regional indices and analytics | `regional-index-mcp` |
